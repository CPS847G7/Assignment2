## Generative Coverage

Install coverage package:

```
pip3 install coverage
```

Run coverage command:

```
coverage run test.py 
```

Output textual report:

```sh
coverage report
```

Generate html coverage report:

```
coverage html
```

Report will be under coverage/index.html
